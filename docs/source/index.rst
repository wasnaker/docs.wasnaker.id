Selamat Datang di Dokumentasi Wasnaker.ID!
===================================

**Wasnaker.ID** adalah aplikasi yang dapat digunakan untuk manajemen sertifikasi K3 oleh pengguna maupun penyedia layanan.
Aplikasi ini dapat diakses di alamat `Wasnaker.ID <https://wasnaker.id/>`_
juga disediakan informasi yang *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

Bagi pengguna layanan sertifikasi K3 silahkan pelajari Bab :doc:`pengguna` untuk informasi berikutnya, atau anda dapat
langsung membaca bagian :ref:`pendafaran` sebagai pengguna.

.. note::

   This project is under active development.
